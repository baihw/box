一个希望能够保持简约，并且具有较强的环境适应性的开发框架。

## 获取

- **maven引用** 
  ```xml
    <dependency>
            <groupId>com.wee0.box</groupId>
            <artifactId>box-{moduleName}</artifactId>
            <version>{version}</version>
    </dependency>
  ```
- **手动编译**  
   下载源码，执行`mvn clean package`。
